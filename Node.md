# 正确的架构边界（最终版）

| 模块                   | 职责                                           |
| ---------------------- | ---------------------------------------------- |
| **ClassTypeValidator** | 类型级语义验证（角色、构造函数、成员、Scope）  |
| **DiGraphValidator**   | 依赖图验证（循环依赖、不可解析依赖）           |
| **Analyzer**           | 用户代码行为验证（禁止手动注入、禁止错误调用） |
| **Generator**          | 生成 Host/User/Scope/Service 代码              |
